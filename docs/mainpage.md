## Introduction

Welcome to the documentation portal for the **C++ Course Template**.  
This repository provides a professional-grade C++ project structure designed specifically for teaching and evaluating C++ programming in an academic environment.

The course template includes:

- a clean separation between **instructor** and **student** code  
- a complete **testing workflow** using **CTest**  
- **coverage instrumentation** using gcov  
- automatic **Doxygen** documentation generation  
- a structured and interconnected **Course Guide**  
- automated validation through **GitHub Actions**

The documentation is organized into multiple layers:

- **High-level topics** inside the **Course Guide**  
- **Support sections** such as:  
  - Local Workflow  
  - Directory Structure  
  - GitHub Workflow  
  - Lesson 0  

Use this portal as the main entry point to explore every component of the course template.
